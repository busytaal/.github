# BusyTaal

BusyTaal builds software at the intersection of human-machine collaboration, semantic systems, and practical engineering.

The public site describes BusyTaal as focused on advanced embedding systems and semantic technologies, with open work centered on voice applications and narrative tooling.

## Projects

### Prepatu

Prepatu is an open-source engine for building reliable, multi-step voice applications. Conversations are defined as YAML state machines so the engine handles routing while models focus on language generation.

- YAML-defined state machines
- Multi-step voice workflow orchestration
- Cloud, self-hosted, or SDK usage
- Browser UI artifact support

Repository: https://github.com/busytaal/prepatu

### BookWriter

BookWriter is a full-stack application for crafting and visualizing complex narratives. It combines a Python backend with an interactive React frontend for plot development and visualization.

- Generative plot development
- Interactive SVG plot visualizer
- Dynamic text editor with chunked loading
- Project-based workflow

Repository: https://github.com/busytaal/bookwriter
